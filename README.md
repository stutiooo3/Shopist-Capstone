# Shopist QA Capstone Automation Framework

An end-to-end test automation framework built using Playwright and Allure Reporting to validate the core functional workflows of the Shopist (shopist.io) platform.

## 🚀 Project Features
* **126 Test Cases** mapped completely across 8 independent functional services.
* **Cross-Browser Engine Validation** running on Chromium, Firefox, and WebKit.
* **Parallel Test Sharding** configured for GitHub Actions CI/CD.
